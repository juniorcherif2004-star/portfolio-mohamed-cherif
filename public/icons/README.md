# Icons Directory

Ce dossier contient les icônes des technologies utilisées dans le portfolio.

## Icônes requises

Pour chaque technologie, ajoutez les fichiers SVG correspondants :

### Frontend
- `react.svg` - Icône React
- `nextjs.svg` - Icône Next.js
- `typescript.svg` - Icône TypeScript
- `tailwindcss.svg` - Icône Tailwind CSS
- `javascript.svg` - Icône JavaScript
- `html5.svg` - Icône HTML5
- `css3.svg` - Icône CSS3

### Backend
- `nodejs.svg` - Icône Node.js
- `python.svg` - Icône Python
- `postgresql.svg` - Icône PostgreSQL
- `mongodb.svg` - Icône MongoDB
- `express.svg` - Icône Express.js

### Tools & DevOps
- `git.svg` - Icône Git
- `docker.svg` - Icône Docker
- `aws.svg` - Icône AWS
- `vercel.svg` - Icône Vercel
- `figma.svg` - Icône Figma

### Mobile
- `reactnative.svg` - Icône React Native
- `flutter.svg` - Icône Flutter

### Testing
- `jest.svg` - Icône Jest
- `cypress.svg` - Icône Cypress

## Sources d'icônes recommandées

- [Simple Icons](https://simpleicons.org/)
- [Devicons](https://devicons.dev/)
- [SVG Repo](https://www.svgrepo.com/)

## Format recommandé

- Format : SVG
- Taille : 24x24px ou 32x32px
- Style : Monochrome ou couleur de la marque
- Vue : Optimisé pour le web

Les icônes manquantes utiliseront automatiquement un fallback avec les initiales de la technologie.
