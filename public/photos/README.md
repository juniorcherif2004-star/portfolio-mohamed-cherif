# Photos Directory

Ce dossier contient les photos personnelles pour le portfolio.

## Photo de profil

- **Nom requis** : `profile.jpg`
- **Emplacement** : `/photos/profile.jpg`
- **Utilisation** : Section Hero du portfolio
- **Taille recommandée** : 400x400px minimum
- **Format** : JPG ou PNG
- **Qualité** : Moyenne (optimisée pour le web)

## Instructions

1. Placez votre photo de profil dans ce dossier
2. Renommez-la en `profile.jpg` si nécessaire
3. Assurez-vous que la photo est claire et professionnelle
4. Le portfolio utilisera un fallback si la photo n'est pas présente

## Fallback

Si la photo n'est pas trouvée, le portfolio affichera :
- Initiales "MJ" sur fond dégradé
- Design moderne et professionnel

## Note

La photo sera automatiquement redimensionnée et optimisée par le navigateur pour s'adapter aux différentes tailles d'écran (mobile, tablette, desktop).
