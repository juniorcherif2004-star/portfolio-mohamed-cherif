# Images Directory

Ce dossier contient les images utilisées dans le portfolio.

## Images requises

### Photo de profil
- **Nom** : `profile.jpg`
- **Taille recommandée** : 400x400px minimum
- **Format** : JPG ou PNG
- **Usage** : Section Hero du portfolio

### Images des projets
- **Nom** : `portfolio.jpg` - Portfolio personnel
- **Nom** : `api.jpg` - API REST Node.js
- **Nom** : `expense.jpg` - Application React
- **Nom** : `business.jpg` - Site web entreprise
- **Nom** : `nestjs.jpg` - Mini-projet NestJS
- **Nom** : `exercises.jpg` - Exercices JavaScript

## Instructions

1. Ajoutez votre photo de profil personnelle dans `profile.jpg`
2. Ajoutez des captures d'écran de vos projets dans les fichiers correspondants
3. Si vous n'avez pas les images de projets, des placeholders seront utilisés

## Taille et optimisation

- **Photo profil** : 400x400px minimum, qualité moyenne
- **Projets** : 800x600px recommandé, compression web
- **Format** : JPG pour les photos, PNG pour les interfaces

Le portfolio utilisera des fallbacks si les images ne sont pas présentes.
